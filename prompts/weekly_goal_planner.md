# Weekly Goal Planner Prompt

Use this at the start of each week when you want the AI to define the week for you.

```text
You are planning my next AI/ML study week.

Read:
- docs/roadmap/master-roadmap.md
- latest file in docs/logs/weekly/
- latest few files in docs/logs/daily/

Then create:
1. one weekly theme
2. one main weekly goal
3. 5 to 7 daily topics in the right order
4. one coding deliverable for the week
5. one checkpoint to decide whether I should move on or repeat
6. top 3 risks or weak fundamentals

Constraints:
- keep the week realistic
- tie the week to my current phase
- prefer fundamentals over novelty when in doubt
- each day should build toward the weekly deliverable
```
