# Daily Commands

## Start the Day

Open `prompts/daily_start.md` and use it with your AI assistant.

## Daily Radar

Open `prompts/daily_radar.md` and capture one high-signal trend in 10 to 15 minutes.

## Teach a Topic

Open `prompts/teach_topic.md` and replace the topic.

## Close the Day

Open `prompts/end_of_day.md` and paste your results.

## Weekly Review

Open `prompts/weekly_review.md` at the end of the week.

## Daily Repository Habit

1. create or update today's file in `docs/logs/daily/`
2. create today's trend note in `docs/radar/daily/`
3. save code in the right folder under `code/`
4. commit with a clear message
5. push to GitHub after the session

## Suggested Commit Style

- `study: add day 1 foundations notes`
- `code: implement logistic regression from scratch`
- `review: add week 1 summary`
