# Master Roadmap

## North Star

Become strong in machine learning, deep learning, and modern LLM systems from first principles through rigorous study, implementation, and evaluation.

## Primary Track

- ML foundations
- deep learning core
- PyTorch training discipline
- NLP and transformers
- modern LLM systems

## Phases

### Phase 1: Foundations

Focus:

- linear algebra intuition
- calculus for optimization
- probability basics
- regression and classification
- loss functions
- overfitting and validation

Suggested sources:

- Google Machine Learning Crash Course
- 3Blue1Brown neural networks
- Deep Learning Book

Outputs:

- linear regression from scratch
- logistic regression from scratch
- notes on gradient descent

### Phase 2: Deep Learning Core

Focus:

- neural networks
- backpropagation
- activations
- optimization
- regularization
- CNNs
- training behavior

Suggested sources:

- DeepLearning.AI Deep Learning Specialization
- Dive into Deep Learning
- PyTorch basics

Outputs:

- MLP on MNIST
- CNN on CIFAR-10

### Phase 3: PyTorch and Training Practice

Focus:

- datasets and dataloaders
- modular model code
- training loops
- validation loops
- metrics
- checkpointing
- experiment debugging

Outputs:

- reusable training pipeline
- debugging checklist

### Phase 4: NLP and Transformers

Focus:

- tokenization
- embeddings
- attention
- transformer blocks
- language modeling
- pretraining and fine-tuning

Suggested sources:

- Stanford CS224N
- Hugging Face Course

Outputs:

- character-level language model
- tiny transformer

### Phase 5: LLM Systems

Focus:

- SFT
- LoRA and PEFT
- quantization
- RAG
- evaluation
- hallucinations and failure analysis
- agents and tool use

Suggested sources:

- Stanford CME 295
- Hugging Face docs

Outputs:

- fine-tune a small model
- evaluate a small LLM workflow

## Operating Rules

- keep one daily log per study day
- keep one weekly review per week
- always pair theory with code
- do not move forward without testing understanding
- ask AI to quiz, explain, and critique code
